# 🏢 النظام المحاسبي الشامل المتطور - Advanced Comprehensive Accounting System

## 🎯 نظرة عامة | Overview

نظام محاسبي شامل ومتطور مطور بأحدث التقنيات (.NET 8 & WPF)، يوفر حلولاً متكاملة ومتقدمة لإدارة العمليات المحاسبية والمالية للمؤسسات من جميع الأحجام. النظام يتضمن **25+ نافذة متطورة** مع ميزات متقدمة مثل **التقارير الذكية**، **مراقبة النظام**، و**النسخ الاحتياطية التلقائية**.

## 🚀 البدء السريع | Quick Start

### 📋 المتطلبات
- **Windows 10/11** (64-bit)
- **.NET 8 Runtime** أو أحدث
- **SQL Server LocalDB** (مثبت مع Visual Studio)
- **4GB RAM** كحد أدنى (8GB مُستحسن)
- **2GB مساحة فارغة** على القرص الصلب

### ⚡ التثبيت والتشغيل السريع

```bash
# استنساخ المشروع
git clone https://github.com/yourusername/advanced-accounting-system.git

# الانتقال إلى مجلد المشروع  
cd advanced-accounting-system

# استعادة الحزم والتبعيات
dotnet restore

# بناء المشروع
dotnet build

# تشغيل التطبيق
dotnet run --project AccountingSystem.WPF
```

### 🎯 التشغيل باستخدام المهام المجهزة

```bash
# بناء النظام
dotnet run --task "Build Accounting System"

# تشغيل النظام مع البناء التلقائي
dotnet run --task "Run Accounting System"
```
